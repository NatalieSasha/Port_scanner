# Port_scanner
# Vulnerability Scanner (Web App)

A lightweight web-based vulnerability scanner built using **Python, Flask, and Nmap**, featuring a clean UI and deployed online for easy access.

---

## Live Demo
👉 https://portscanner-production.up.railway.app/

---

##  Features

- 🔍 Scan target IP for open ports (1–1024)
- ⚠️ Identify potentially risky services
- 💻 Interactive web interface using Flask
- 🎨 Clean, minimal UI design
- 🌐 Deployed and accessible online

---

## ⚠️ Note

Due to cloud restrictions, **live port scanning (Nmap)** is not supported in deployed environments.

👉 The hosted version runs in **demo mode**  
👉 Full scanning functionality works **locally**

---

## 🛠 Tech Stack

- Python  
- Flask  
- Nmap  
- HTML / CSS  
- Git & GitHub  
- Railway (Deployment)
